# Practica 2: Herencia y reutilizacion de codigo en PHP

## Objetivo de la practica
Implementar herencia mediante la extension de clases, reutilizando atributos y metodos de una clase base en PHP.

## Explicacion de la herencia aplicada
La herencia es un pilar fundamental de la pogramacion orientada a objetos que permite crear una nueva clase basada en una clave existente.

- **Clase base (padre)**: "Usuario" - Contiene atributos y metodos genericos.
