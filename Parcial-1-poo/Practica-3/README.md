# Practica 3: Sistema de Usuarios con Validaciones y Excepciones

## Descripcion del sistema
Sistema orientado a objetos desarrollado en PHP que implementa una jerarquia de usuarios con validaciones robustas y manejo de excepciones. El sistema simula un entorno profesional donde los datos de entrada son validados antes de crear objetos lanzando excepciones cuando no cumplen con los criterios establecidos.
